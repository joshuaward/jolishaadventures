# Jo & Alisha Adventures

A scroll-snap presentation for our European adventure — built with React and Vite, hosted on GitHub Pages.

## Live site

After GitHub Pages is enabled: **https://joshuaward.github.io/jolishaadventures/**

## Local development

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Deploy

Pushes to `main` automatically deploy via GitHub Actions (`.github/workflows/deploy.yml`).

To enable Pages the first time:

1. Open **Settings → Pages** on the repo
2. Set **Source** to **GitHub Actions**

## Legacy HTML

The original standalone file is kept as `european-adventure-for-alisha.html` for reference.
